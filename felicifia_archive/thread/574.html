Re: Felicifia backups

by JamesEvans on 2012-03-11T23:04:00

The best way to backup the site is to grab a copy of the SQL database. You can use the phpmyadmin interface here:
https://www.knightsbridge.net/phpmyadmin/
If someone emails me I can send the DB password or you can FTP it directly from the config file in the WP directory.
The SQL DB is normally backed up in a weekly backup (and that backup is itself copied to an offsite location). But it is safest to make another backup as well so I'd recommend a manual export every so often via phpmyadmin.

Instructions:
https://codex.wordpress.org/Backing_Up_Your_Database
